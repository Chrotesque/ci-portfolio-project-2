Help

Simon Game is a game of memorizing the order of buttons pressed by the computer, then correctly repeating the order yourself.

Both you and the computer take turns to press the game buttons, always starting with the computer.

The sequence of buttons pressed grows with each turn and remains unchanged until the end of the round.

Once you have replicated the entire sequence correctly, the round is won and a new one starts.

Upon making a mistake, the game is either over OR the current computer turn is repeated - depending on the difficulty.

There are 4 difficulty levels:

  • Easy: 3 buttons, slow speed, multiplier of 0
  • Normal: 4 buttons, normal speed, ramps up, multiplier of 0
  • Hard: 5 buttons, fast speed, strict on, ramps up, multiplier of 0
  • Custom: customized through the settings menu, current multiplier of 0

Points are given for each correct button press (0), a fully completed turn (0) and for winning a round (0). However, these are the base values and multiplied by the multiplier seen above.

A few settings explained:

  • Game Settings don't affect the multiplier of the custom difficulty
  • Easy, Normal and Hard have fixed settings each and cannot be changed
  • Advanced Game Info displays additional game info in the middle during rounds
  • Strict Mode controls if making a mistake will end the game
  • Ramp Up will increase the difficulty each round, affecting sequence length and speed
  • Chaos Mode is a surprise, give it a try!
